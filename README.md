<h1>¿Cómo?</h1>
<h2>Una Introducción a la Epistemología del Saber-Cómo</h2>
</div>

Este es un libro de texto introductorio a la epistemología del saber-cómo o
conocimiento práctico. Está pensado como una introducción a la epistemología
contemporánea, tomando el saber-cómo como hilo central.

Incluye una discusión de:

1) Metodologías filosóficas,
2) La disputa entre intelectualistas y anti-intelectualistas sobre el saber-cómo,
3) La filosofía de la mente relevante al saber cómo,
4) La posibilidad de saber-cómo artificial y colectivo,
5) El problema de la injusticia epistémica y los vicios epistémicos que podrían asociarse al saber-cómo
6) Enfoques etnoepistemológicos sobre el saber-cómo.

Este libro es parte del proyecto ANID/FONDECYT de Postdoctorado #3220017
"Saber-cómo: preguntas, intentos y maneras", a cargo de [Felipe Morales
Carbonell](https://fmoralesc.github.io).

# Instrucciones

El libro ha sido preparado usando el sistema de procesamiento de texto [typst](https://typst.app). Para compilarlo,
es necesario tener typst instalado localmente:

```
$ typst main.typ
```

Los archivos están organizados de la siguiente manera:

| Ruta:           | Descripción                                                                         |
|-----------------|-------------------------------------------------------------------------------------|
| main.typ        | Archivo principal, contiene la configuración general del documento y la estructura  |
| cover.typ       | La tapa del libro                                                                   |
| frontmatter.typ | Sección inicial del libro, antes de los capítulos                                   |
| backmatter.typ  | Sección final del libro, incluye la bibliografía                                    |
| utils.typ       | Algunos componentes genéricos (cajas de información, preguntas)                     |
| logic.typ       | Facilidades para preparar argumentos y fórmulas lógicas                             |
| **bib/**        | Archivos de bibliografías                                                           |
| **chapters/**   | Capítulos                                                                           |
| **fonts/**      | Fuentes usadas (Minion Pro, Jost, Adobe Caslon Pro)                                 |
| **images/**     | Imágenes                                                                            |
